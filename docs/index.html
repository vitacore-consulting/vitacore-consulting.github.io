ЭДО ОМС - решение для организации юридически значимого документооборота в сфере обязательного медицинского страхования.

Возможности программы

ЭДО ОМС упрощает работу с документами как внутри организации, так и при взаимодействии с внешними контрагентами. Решение помогает организовать создание, передачу, согласование и утверждение документов, подписанных электронной подписью.

  • Создание карточки документа

  • Прохождение документа по маршруту, подписание документа КЭП сторон

  • Выгрузка исходного документа и подписей сторон к нему

  • Управление учетными данными пользователей
